ABSTRACT
This study aimed to study the effect of extraterrestrial radiation on low-level cloud base height at Ubon Ratchathani Airport by using global radiation intensity from a pyranometer as a representative of extraterrestrial radiation. and the data from the ceilometer to find the relationship and equation between the height of the cloud base using 10 years of data between 2015 â€“ 2025 By using the Multiple Regression Analysis method, it was found that the height of the low-level cloud base at Ubon Ratchathani Airport has a relationship with the intensity of total solar radiation. And the relationship equation is LCBH= 4.5X - 271.6 where X = Global Radiation in the Ubon Ratchathani Airport area (MJ/m2) and LCBH = Low level Cloud Base Height (ft) with an efficiency of 37.0%. The study also found that there are many relationships that are useful for aviation meteorology, such as the highest cloud base height during the cold season, the lowest cloud base height during the rainy season, and the lowest cloud base height during the day, which is lower than at night.
